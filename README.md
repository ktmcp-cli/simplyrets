# @ktmcp/simplyrets

MCP server for SimplyRETS

## Installation

```bash
npm install -g @ktmcp/simplyrets
```

## Usage

Add to your MCP settings:

```json
{
  "mcpServers": {
    "simplyrets": {
      "command": "mcp-simplyrets"
    }
  }
}
```
